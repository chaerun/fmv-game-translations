# Love Chase - Translation Patch

Thank you for downloading this community translation patch!

## ⚠️ Important

Please ensure the game is **fully closed** before installing.

## 🛠️ Installation Instructions

1. **Extract the ZIP file** you just downloaded to a folder on your computer. You should see a folder named `Love Chase_Data` alongside this README file.
2. **Open Steam**, go to your Library, and right-click on **Love Chase**.
3. Select **Manage** -> **Browse local files**. This will open the game's original installation folder (typically located at `...\steamapps\common\LoveChase`).
4. **Copy** the `Love Chase_Data` folder from your extracted ZIP.
5. **Paste** it directly into the `steamapps\common\LoveChase` folder.
6. When your operating system asks if you want to replace or overwrite existing files, choose **Yes / Overwrite**.
7. Launch the game and enjoy!
