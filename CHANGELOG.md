## [love-chase/v1.1.0] - 2026-06-19

### 🌐 Translation Updates

- Corrected the translations regarding the girls in the `Behind The Scenes` menu.
- Standardized the Affection Level translations to match the developer's original asset names

  ```text
  Stranger → Friend → Like → Partner → Love
  ```

- Standardized the girls' names for consistency across all dialogue and menus:
  - Monroe / Menglu / Meng Lu → **Meng Lu**
  - Suxin / Su Xin → **Su Xin**
  - Wanlin / Wan Lin → **Wan Lin**
  - Jiangyan / Jiang Yan → **Jiang Yan**

### 🖥️ UI & Formatting Fixes

- Fixed the font size on the `Title Screen`.
- Fixed a visual UI issue in the `Behind The Scenes` menu.
- Adjusted text sizes to ensure all text is fully visible and does not clip or overflow within the UI components.
- Added extra spacing around colored text to improve readability.
- Replaced curly apostrophes (’) with straight apostrophes (') to resolve a rendering issue that caused unwanted extra spacing after the characters.
